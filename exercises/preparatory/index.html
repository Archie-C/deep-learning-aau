Exercise 1

Basic Probability

Scenario

An agentic AI system writes code blocks using two models, Model A and Model B. A code block is selected at random.

  • Model A is used 60% of the time and has a 5% chance to produce a code block that contains an error.
  • Model B is used 40% of the time and has a 2% chance to produce a code block that contains an error.

Tasks

  1. Find the probability that the selected code block contains an error.
  2. Given that the selected code block contains an error, find the probability that it was produced by Model A.
  3. Find the probability that a code block is either:
    • produced by Model A, or
    • defective.
