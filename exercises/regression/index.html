Regression exercises

These exercises are for the regression section of the course. They are mostly designed to be solved with pen and paper. Obviously, I cannot stop you from using AI tools to solve them. However, I encourage you to solve them yourself first as they WILL help in the exam. The exam questions will be similar in style.


Please bring your solutions to the next session, we will discuss them together and then I will assess them and provide feedback.

Question 1

Derivation

Scenario

We would like to predict the price of a house based on its size (in square metres) and the number of bedrooms. We have a dataset of \(N\) houses, where we have the size \(x\), the number of bedrooms \(z\), and the price \(y\). We want to fit a linear regression model of the form:

\[ y = w_0 + w_1 x + w_2 z \]

Where \(w_0\) is the intercept, \(w_1\) is the coefficient for the size, and \(w_2\) is the coefficient for the number of bedrooms.

Task

Derive the closed-form solution for the parameters \(w_0, w_1, w_2\). Provide each step of the derivation.


Question 2

Basis Functions

Scenario

I would like to model the relationship between the air quality index (AQI), temperature, and the number of hospital admissions in a given city. I have a dataset of \(N\) days, where for each day \(i\), i have the AQI \(x_i\), the temperature \(T_i\), and the number of hospital admissions \(y_i\). My aim is to predict the number of hospital admissions based on the AQI and temperature using a regression model.

Tasks

  1. Explain why a standard linear model may not be sufficient for this problem.
  2. Define a suitable feature mapping \(\phi(x, T)\) that includes non-linear effects.
  3. Show that the resulting model is still linear in the parameters.


Question 3

Generalisation

Scenario

A student fits two regression models to the housing dataset as before:

  • Model A: \(y = w_0 + w_1 x + w_2 z\)
  • Model B: uses a polynomial basis function up to degree 5 for both features.

Model B achieves a much lower training error than Model A.

Tasks

  1. Explain why you expect Model B to have a lower training error than Model A.
  2. Explain why Model B might perform worse on unseen data.
  3. Describe one method to select between Model A and Model B in practice.
